# CATIA V5 Fundamentals & Multi-Component Design
A comprehensive CATIA V5-R21 project covering 5 core modules and multiple real-world component designs — completed as part of B.Tech Mechanical & Automation Engineering at IGDTUW.

## Modules covered
**1. Sketcher module**
Created precise 2D profiles and constrained sketches as the foundation for all 3D models.
**2. Part Design module**
Designed complex 3D parts using pads, pockets, shafts, fillets, chamfers, and patterns. Components include a Hook, U-Profile, Lever Spring, Chair, and mechanical brackets.
**3. Assembly module**
Assembled multiple individual parts into complete products using constraints and connections. Designed a full Pressing Tool assembly with guide pillars, bushes, and plates.
**4. Surfacing module**
Created complex surface geometry with no defined thickness using extrudes, blends, offsets, and intersections. Designed a Bottle and Cane using surface tools.
**5. Sheet metal module**
Designed foldable/unfoldable sheet metal parts including enclosures and covers using wall features, cutouts, and mirror operations.

## Components designed
| Component | Module | Description |

- 3D parametric modelling in CATIA V5
- Feature-based part design (pad, pocket, shaft, mirror, pattern)
